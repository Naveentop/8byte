# ⚖️ Horizontal Pod Autoscaler (HPA) – CPU & Memory Based Scaling

## 📌 Overview
This project demonstrates how to configure Kubernetes Horizontal Pod Autoscaler (HPA) to automatically scale application pods based on **CPU and Memory utilization**.

HPA helps ensure:
- High availability during traffic spikes
- Efficient resource utilization
- Cost optimization
